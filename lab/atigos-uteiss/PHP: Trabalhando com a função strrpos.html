PHP: Trabalhando com a função strrpos

Publicado em: 18 de outubro de 2012Categoria: PHPTag(s): strrpos()
    Obrigado! Você é a 1.563ª pessoa que visita este post.

    Definição:
    strrpos() – Encontra a posição da última ocorrência de uma string.

    Maiores detalhes sobre essa função, clique aqui.

    Colaboração: Valter Ferreira Martins

    Idéia de utilização:
    Imagine que você tenha empresas filiais e precisa filtrar o conteúdo de um sistema pelo ip.

    Exemplo:

    <?
$ip = '192.168.1.27'; // filial em Paraná
$ip = '192.168.30.30'; // filial no Goiás
 
$pos = strrpos( $ip, "." );
 
if ($pos == 9){
  echo "Você é do Paraná";
}else{
  echo "Você é de Goiás";
};
?>

    Veja que existe uma diferença entre os ips. Antes de chegar ao último ponto (.), um ip possui 1 número(1) e o outro 2 números (30).

    192.168.1.27 -> strrpos = 9

    192.168.30.30 -> strrpos = 10
