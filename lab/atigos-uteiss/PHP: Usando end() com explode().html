PHP: Usando end() com explode()

Publicado em: 10 de maio de 2011Categoria: PHPTag(s): end(), explode

    Usando end() para pegar a parte final de uma string separada pelo explode().

    <?php
   $url_pagina = "http://www.site.com.br/paginas/contato";
   echo end(explode("/",$url_pagina));
?>

    Resultado:
    contato
