Diversas funções para vídeos no youtube (gerar código <embed>, o

criado por Jefrey em 22/01/2011 3:18pm
1. Transforma URL do youtube em um código HTML de inclusão do vídeo completo.
2. Obtém título original do vídeo.
3. Obtém descrição original do vídeo.

Uso:
string youtube ( string $video [, int $largura [, int $altura ]] )
string youtube_title ( string $video )
string youtube_description ( string $video )

Caso você queira obter todos os dados do vídeo (título, descrição e código de inclusão), use a versão ALL. O processo se tornará mais veloz.
array youtube ( string $video )
Ele retorna um vetor com os ítens 'title', 'desc' e 'code'.
Exemplos (função ALL):
// PHP < 5.4.0
$video = youtube("http://www.youtube.com/watch?v=123456&feature=embed");
echo $video['code'];
echo $video['title'];
echo $video['desc'];

// PHP >= 5.4.0
echo youtube("http://www.youtube.com/watch?v=123456&feature=embed")['code'];
echo youtube("http://www.youtube.com/watch?v=123456&feature=embed")['title'];
echo youtube("http://www.youtube.com/watch?v=123456&feature=embed")['desc'];
Obs.: Caso você queira obter todos os 3 dados, faça como se estivesse numa versão menor que 5.4.0.
